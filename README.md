# Meta-analysis-of-OAC-after-ICH
Reproducible MCMC code and data for the meta-analysis of OAC after ICH.
This repository contains the code and data for reproducing the MCMC analysis.

## Files
- `model.stan`: Stan model code.
- `run.R`: R script to run the analysis.
- `data.csv`: Input trial data.

## Instructions
Open `run.R` in RStudio and run the script. The results will be saved in the `results/` folder.
